# 간략 소개
이 프로젝트는 Unity 엔진과 C#을 기반으로 개발된 3D 미로 탈출 게임입니다.  
트리 기반의 알고리즘(DFS)을 활용하여 랜덤한 미로를 생성하고,  
플레이어는 WASD 키로 미로에서 목적지에 도달하여 탈출 하는 게임입니다.


# 개발 환경
○ Unity 2022.3.30f1  
○ c#  
○ Window
